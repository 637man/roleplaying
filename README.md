# Roleplaying Resources

This repository was set up to collect various resources for role-playing games, mainly Markdown conversions of existing PDFs. Main reason is turning them into actual structured documents with hyperlinking for quicker reference when both playing and GM-ing or DM-ing, as well as eventual translation into Czech for those without ESL certificates. Having to constantly list or scroll through would disrupt the game pace. Also it's more accessible this way, namely my eyes cannot stand the white background of the PDFs.

No license is implied, all rights belong to original authors.

Current focus lies on resurrecting GURPS for the purpose of creating an escapological campaign that will polemize with many sociological assumptions in there in the context of modern-day consensual power exchanges. Note that only Lite is redistributable, and only unchanged, yet translations are encouraged. Some d20 SRDs will likely be imported as well for ontological cross-breeding.

Czech translations are likely going to respect existing DrD (Dračí doupě) and JaD (Jeskyně a Draci) conventions for familiarity. Note that GURPS is unavailable in Czech, despite only requiring spare d6 readily available in state institutional dungeons and some paper(work). The Basic Set is available in Russian in a typical unofficial Eastern European way, though.
